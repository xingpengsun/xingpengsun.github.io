  • Multiple Artistic Style Transfer Neural Network

    Python

    Keras

    Tensorflow

    Multiple Style transfer is a neural network application designed to synthesize an image that inherits the content of a content image while preserving a medley of all styles of the style images. Based on the VGG16 model, I customize the loss function and transfer two styles onto one content image simultaneously.



  • Fantasy Football Team Drafting Optimization

    Python

    Julia

    Fantasy football is a widely played game in America where a set of league “owners” draft actual National Football League players to their fantasy teams and accrue points each week based on the on-field performance of those players. This project is a Linear Programming model which optimized the snake draft strategy based on player value, player position, player projected score, league regulation, etc. Projected Opt results are 80% alike to the actual draft decision.
